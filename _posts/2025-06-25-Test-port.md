## Windows
Using Powershell you can test open ports using this command.
```
test-netconnection -ComputerName 4.2.2.1 -Port 53
```

## Mac
If the installation of `netcat` is needed 
```
sudo apt-get install netcat
```

Command
```
nc -vz <IP> <PORT>
```
