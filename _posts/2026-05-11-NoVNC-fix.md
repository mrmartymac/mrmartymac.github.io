When Docker / NoVNC Sessions are failing try to restart Docker

```bash
systemctl restart docker
```

If this does not resolve the issue, restarting MOAI may be needed.

```bash
cd /u/moai/server
./moai.py status
./moai.py stop
./moai.py start
```
