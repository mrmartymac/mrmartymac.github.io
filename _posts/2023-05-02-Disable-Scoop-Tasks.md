Connect to the postgres database and run the command shown below to disable active tasks.

Open PSQL
```bash
psql -U postgres
```

Connect to the Scoop database
```bash
\c scoop_db
```

Run query to disable active tasks.
```sql
UPDATE tasks
SET enabled = 'false'
WHERE enabled = 'true';
```

Disconnect from the database  
`\q`
